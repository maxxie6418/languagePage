## 三、项目结构（当前状态）

```
languagePage/
├── backend/
│   ├── server.js           # Express 入口，注册全部路由
│   ├── config/database.js  # SQLite 连接配置
│   ├── middleware/auth.js   # JWT 验证中间件
│   ├── routes/             # 路由层
│   │   ├── auth.js         # 认证
│   │   ├── words.js        # 单词
│   │   ├── knowledge.js    # 知识点
│   │   ├── ebbinghaus.js   # 艾宾浩斯
│   │   ├── quiz.js         # 测验
│   │   ├── ai.js           # AI 代理
│   │   └── admin.js        # 管理员
│   ├── controllers/        # 控制层
│   └── models/             # 数据模型
├── public/
│   └── index.html          # SPA 单页应用（CSS + JS 内联）
├── database/
│   ├── migrations/init.js  # 数据库表创建
│   └── seeds/seed.js       # 种子数据（497词/38知识点/13题）
├── data/learning.db        # SQLite 数据库文件（git忽略）
├── docs/                   # 文档
├── 启动.bat                 # Windows 一键启动菜单
├── package.json
└── AGENTS.md               # OpenCode 指令文件
```

## 四、启动方式

```bash
npm install
cp .env.example .env   # 编辑 JWT_SECRET
npm run init-db         # 创建数据库表
npm run seed-db         # 导入种子数据
npm run dev             # 开发模式启动 → http://localhost:3000
```

或双击 `启动.bat` 使用菜单操作。

## 五、已实现的关键功能详情

### 5.1 用户系统
- 注册/登录（JWT 7天有效期）
- 第一个注册用户自动为管理员
- 最多 50 用户
- 游客模式可浏览学习内容
- 登录后学习记录按 user_id 隔离

### 5.2 单词掌握状态
- 三态循环：unmastered → learning → mastered
- 标记 mastered 自动生成艾宾浩斯 6 阶段复习计划
- 批量选择后批量标记状态
- 单词检测模式（隐藏中文/英文）

### 5.3 艾宾浩斯复习
- 6 阶段间隔：1/2/4/7/15/30 天
- 今日待复习列表（首页展示）
- 复习确认弹窗：已掌握（推进下一阶段）/ 未掌握（重置为 unmastered）
- 提醒开关

### 5.4 AI 功能（需管理员配置 DeepSeek API Key）
- 通用对话（AI 浮窗）
- 单词深度解析（词根词缀、搭配、例句）
- 语法拆解（长难句结构分析）
- 作文批改（四维评分）

### 5.5 测验系统
- 快速测验（选题型 + 题数）
- 随机出题 → 逐题作答 → 提交判卷 → 显示结果
- 错题自动录入错题本
- 错题回顾（按题型筛选）

### 5.6 管理员功能
- 用户列表/重置密码/删除用户
- AI 配置（API Key / Base URL / 模型）
- 删除用户时级联删除所有学习数据
